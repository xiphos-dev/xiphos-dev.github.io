Reconocimiento de estrategias en StarCraft II

Memoria para optar al título de Ingeniero Civil en Informática. Diseñé múltiples representaciones diferentes para los datos de entrenamiento para redes nueronales densas, recurrentes y convolucionales. Cada tipo fue puesto a prueba para determinar al mejor candidato. El objetivo es encontrar una o más soluciones factibles para el problema de reconocimiento de estrategias, en particular, estrategias usadas por jugadores humanos. Este problema resulta relevante dado a que se relaciona con las múltiples aristas del desafío de crear un agente artificial capaz de navegar las complejidades de juegos de estrategia sofisticados.
